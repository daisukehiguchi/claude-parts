# ClaudeParts. 006 / 初診の方向けフローチャート

医院・クリニック向けの「初診の方向けフローチャート」パーツです。来院から会計までの流れを、各ステップの持ち物・所要時間つきで案内します。チェックを入れて進めば、来院中の進行ガイドとしても使えます。

デザインは005番「診療時間・休診日カレンダー」と同じ医院向けトークン（清潔感のあるティール×白ベース、Noto Sans JP）を継承しています。

## 特徴

- **5ステップの来院フロー**：予約 → 受付・問診票 → 問診・検査 → 診察 → 会計
- **持ち物・所要時間のメモ**：各ステップにチップ形式で表示
- **チェック式の進行ガイド**：番号をタップすると完了マークに切り替わり、タイトルに取り消し線
- **進捗バー＆残り時間表示**：チェックした分だけ進捗バーが伸び、残りの目安時間を自動計算
- **持ち物まとめ**：カード下部に必要な持ち物を一覧表示
- **リセットボタン**：チェックを一括で戻せます

## カスタマイズポイント

### ステップの内容・所要時間
`index.html` の `.fc-step` を編集してください。各 `<li class="fc-step" data-minutes="10">` の `data-minutes` が所要時間（分）で、合計時間の自動計算に使われます。所要時間の目安がないステップは `data-minutes="0"` としてください。

### 持ち物まとめ
`.fc-items__list` 内の `<li>` を編集することで、カード下部の持ち物一覧を変更できます。

### 色・フォント
005番と同じ `:root` トークンを使用しています。ClaudeParts.の方針上、業種が変わるまでは同じトークンを引き継ぎます。

```css
--bg: #eef4f3;
--surface: #ffffff;
--primary: #1f7a6c;
--primary-soft: #e1f3ef;
--amber: #c98a3d;
--amber-soft: #faf0e0;
--coral: #c4665c;
--coral-soft: #f6e8e6;
--ink: #1f2a28;
--muted: #6d7d7a;
--line: #e2eae8;
```
