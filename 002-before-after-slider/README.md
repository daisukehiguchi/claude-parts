# 002 / Before / After スライダー

施工の「変化」をひと目で伝える、Before/After比較スライダーです。

## コンセプト

001の施工実績カードが"記録"だとすれば、こちらは"変化を測る"パーツです。
中央のつまみを巻尺のグリップに見立て、境界線は目盛りのような点線にしています。

スライダーを動かすと、位置に応じて「改修前」「改修後」のタグがふわっと濃淡を変えながら現れます。
配色・書体は001と共通のトークンを使い、シリーズとしての一貫性を持たせています。

## 使用技術

- HTML / CSS / JavaScript（フレームワークなし）
- `<input type="range">` をベースにした実装のため、キーボード操作・スクリーンリーダーにも対応
- After画像は `clip-path` で表示範囲を制御し、CSSカスタムプロパティ `--pos` で位置を一元管理

## ファイル構成

```
002-before-after-slider/
├── index.html   … スライダー2件のデモ
├── style.css    … スライダー・カードのスタイル一式
└── script.js    … range入力の値を--posに反映するだけの最小限のJS
```

## 使い方

1. `.ba-slider__img--before` / `.ba-slider__img--after` 内のプレースホルダーSVGを、実際の施工写真の `<img>` に差し替えてください（両方とも同じ構図・同じ画角の写真を使うのがコツです）
2. `<input type="range">` の `value`（初期表示位置）と、`style="--pos:◯◯"`（同じ数値）を揃えてください
3. `.ba-card__title` や `.ba-card__eyebrow` は001と同じ形式なので、実績カードと対にして使えます

## 対応

- タッチ / マウス / キーボード（矢印キー）操作に対応
- `prefers-reduced-motion` 設定時はタグのフェードアニメーションを無効化
