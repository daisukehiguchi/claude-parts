# ClaudeParts. 007 / 院長・スタッフ紹介カード

医院・クリニック向けの「院長・スタッフ紹介カード」パーツです。院長プロフィールをメインに大きく紹介し、スタッフはタブ切り替えで紹介するハイブリッド構成になっています。

デザインは005・006番と同じ医院向けトークン（清潔感のあるティール×白ベース、Noto Sans JP）を継承しています。

## 特徴

- **院長プロフィール（メイン）**：氏名・役職・経歴リスト・院長メッセージを大きく表示
- **スタッフ紹介（タブ切り替え）**：スタッフの名前タブをクリックすると、写真（アイコン）・氏名・役職・コメントがフェード切り替え
- **アバターはSVGアイコン**：実写真を使わなくても導入できるプレースホルダー仕様（差し替え前提）
- レスポンシブ対応（〜400px幅でも崩れないレイアウト）

## ファイル構成


## カスタマイズポイント

### 院長情報
`index.html` の `.pc-doctor` セクション内、氏名・役職・`.pc-doctor__history` の経歴リスト・`.pc-doctor__message` のメッセージ文を差し替えてください。

### スタッフの追加・変更
`.pc-tabs` 内にタブボタンを追加し（`data-target` に対応するパネルIDを指定）、`.pc-staff__panel` 内に同じIDを持つ `.pc-staff__content` を追加してください。JS側の変更は不要です。

### 写真の差し替え
`.pc-avatar` 内のSVGアイコンを `<img>` タグに置き換えれば、実際の写真を表示できます。その場合 `.pc-avatar` に `overflow: hidden` を追加し、画像を `object-fit: cover` で表示するとバランスが取りやすいです。

### 色・フォント
005・006番と同じ `:root` トークンを使用しています。ClaudeParts.の方針上、業種が変わるまでは同じトークンを引き継ぎます。

```css
--bg: #eef4f3;
--surface: #ffffff;
--primary: #1f7a6c;
--primary-soft: #e1f3ef;
--amber: #c98a3d;
--amber-soft: #faf0e0;
--coral: #c4665c;
--coral-soft: #f6e8e6;
--ink: #1f2a28;
--muted: #6d7d7a;
--line: #e2eae8;
```
