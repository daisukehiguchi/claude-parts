# ClaudeParts. 005 / 診療時間・休診日カレンダー

医院・クリニック向けの「診療日カレンダー」パーツです。月間カレンダー上で診療日・休診日を色分け表示し、日付をクリックするとその日の診療時間の詳細が表示されます。

## 特徴

- **月間カレンダー表示**：前月・翌月ボタンで月を切り替え可能
- **4種類のステータス色分け**
  - 診療日（通常）
  - 午前診療のみ
  - 休診日（日曜・祝日など定休）
  - 臨時休診（学会・研修などの不定休、右上に赤ドットで強調表示）
- **日付クリックで詳細表示**：カレンダー下部に選択日の診療時間をテキスト表示
- **本日パルス表示**：当日のセルに小さな鼓動アニメーションのドットが点灯
- **通常診療時間の一覧表**：曜日ごとの午前・午後診療時間をカレンダー下に併記

## カスタマイズポイント

### 医院名・タブ文言
`index.html` 内の `.cc-card__title`（医院名）と `.cc-card__tab`（左上のタブ文言）を差し替えてください。

### 診療日ルール
`script.js` の `getStatus(date)` 関数内で、曜日（`date.getDay()`：0=日〜6=土）ごとのステータスを定義しています。実際の診療体制に合わせて `open` / `am` / `closed` の判定と表示文言を調整してください。

### 臨時休診日
`script.js` の `addOverride(offsetDays, label)` はデモ用に「今日から○日後」で臨時休診を設定しています。実運用では日付を直接指定する形（例：`irregularOverrides.set('2026-6-15', '臨時休診（設備メンテナンスのため）')`、月は0始まりの点に注意）に置き換えるか、サーバー側のデータを読み込む形に拡張してください。

### 通常診療時間テーブル
`index.html` の `.cc-weekly__table` 内の曜日・時間表記をそのまま書き換えるだけで反映されます。

## デザイントークン

クリニックらしい清潔感を意識し、白ベースの背景に落ち着いたティール（診療中）とコーラル（休診・注意）のみを差し色にした配色にしています。

```css
--bg: #eef4f3;
--surface: #ffffff;
--primary: #1f7a6c;
--primary-soft: #e1f3ef;
--amber: #c98a3d;
--amber-soft: #faf0e0;
--coral: #c4665c;
--coral-soft: #f6e8e6;
--ink: #1f2a28;
--muted: #6d7d7a;
--line: #e2eae8;
```

フォントは見出し・本文ともに Noto Sans JP を使用し、太さ（900 / 700 / 500）で情報の階層を作っています。カード上部の脈波（パルス）ラインと、本日セルの鼓動アニメーションが、このパーツ固有のサイン要素です。
