# ClaudeParts. 009 / 物件詳細カード

不動産向けの「物件詳細カード」パーツです。間取り・価格帯を中心に、物件のスペックと簡易間取り図をまとめて表示します。

医院カテゴリ（005〜008）から業種が変わったタイミングで、デザインを刷新しました。黒・ネイビー×ゴールドの上質・高級感のある配色に、シャープな角のレイアウトを組み合わせています。フォントは医院カテゴリと同じくNoto Sans JPを継続していますが、太さと文字間でメリハリをつけています。

## 特徴

- **スカイラインのライン画**：写真の代わりに、ゴールドのラインアートで建物群を表現（実写真に差し替え可能）
- **お気に入りボタン**：カード右上のハートアイコンでON/OFF切り替え
- **価格の強調表示**：大きなゴールド文字で価格帯を表示
- **スペック一覧**：間取り・専有面積・所在階／階建・築年数・最寄り駅・向きを2列グリッドで表示
- **簡易間取り図**：CSS Gridで3LDKの部屋割りを表現したシンプルな図解
- **問い合わせCTAボタン**：ゴールドの縁取り、ホバーで塗りつぶし

## カスタマイズポイント

### 物件情報
`index.html` 内の `.pd-title`（物件名）、`.pd-address`（住所）、`.pd-price__value`（価格）、`.pd-specs__item` 各項目を差し替えてください。

### 間取り図
`.pd-floorplan__diagram` は `grid-template-columns` / `grid-template-rows` と `.pd-room--*` の `grid-column` / `grid-row` で部屋割りを組んでいます。間取りが変わる場合はこの2つを調整してください（例：4LDKにする場合は列・行を増やす）。

### 写真への差し替え
`.pd-visual` 内の `<svg class="pd-skyline">` を `<img>` に置き換えれば、実際の外観写真を表示できます。

フォントはNoto Sans JP（400/500/700/900）。見出しや価格は900、ラベル類は700に letter-spacing を加えて引き締めています。
