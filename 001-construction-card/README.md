# 001 / 施工実績カード

建設業向けサイトの「施工実績」ページでよく使う、実績1件分を表示するカードコンポーネントです。

## コンセプト

建設業の実績紹介は、写真と数字（所在地・工期・規模）が並ぶだけになりがちで、
どのサイトを見ても同じような印象になってしまいます。

このカードでは、

- カード＝クラフト紙の台帳
- ページ背景＝青焼き（ブループリント）の方眼

というコントラストをつけて、"工事の記録"らしい質感を出しています。
所在地・工期・規模・施工会社の並びは、見積書や仕様書でおなじみの点線（ドットリーダー）表記にしました。

右下の「検収済」ハンコは、クリック（またはホバー）すると軽く押印するアニメーションが入ります。
工事完了後に検収印を押す、という実際の業務動作をそのままUIのギミックにしています。

## 使用技術

- HTML / CSS / JavaScript（フレームワークなし、素のまま）
- Google Fonts：見出しに `Shippori Mincho`、本文に `Zen Kaku Gothic New`
- アニメーションは CSS Animation + IntersectionObserver のみ（軽量）

## ファイル構成

```
001-construction-project-card/
├── index.html   … カード3枚のデモ
├── style.css    … カードのスタイル一式
└── script.js    … スクロール出現・押印アニメーション
```

## 使い方

1. `.jc-card` のブロックごと複製すれば、実績カードを何枚でも追加できます
2. `.jc-card__photo` 内の `.jc-card__placeholder`（破線＋アイコン）を、実際の施工写真の `<img>` タグに差し替えてください
3. `.jc-card__tag` のテキスト（新築工事／改修／外構工事など）は自由に変更可能です
4. 所在地・工期・規模・施工会社は `<dl class="jc-card__specs">` 内の `<dd>` を書き換えるだけです

## 対応

- レスポンシブ対応（グリッドは `auto-fit` で自動折り返し）
- `prefers-reduced-motion` 設定時はアニメーションを無効化
- キーボード操作時のフォーカスリング対応済み
