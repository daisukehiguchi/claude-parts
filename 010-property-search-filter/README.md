# ClaudeParts. 010 / 物件検索フィルターUI

不動産向けの「物件検索フィルターUI」パーツです。「駅近」「新築」「ペット可」などの条件タグを複数選んで絞り込む、タグ選択型の検索フィルターです。Phase 1（001〜010）の最後のパーツになります。

デザインは009番「物件詳細カード」と同じ、黒・ネイビー×ゴールドの上質なトーンを継承しています。

## 特徴

- **クイック選択（単一選択）**：エリア／価格帯／間取りをそれぞれ1つずつ選択
- **こだわり条件タグ（複数選択）**：駅徒歩5分以内・新築・ペット可など8種類のタグをトグル選択
- **選択中の条件バー**：選んだ条件がチップで一覧表示され、×で個別に解除可能
- **該当件数の目安を自動計算**：選んだ条件に応じて「検索する」ボタン内の該当件数がリアルタイムに更新（デモ用の簡易計算。実運用ではAPIの検索結果件数に置き換えてください）

## ファイル構成

```
010-property-search-filter/
├── index.html   … マークアップ
├── style.css    … スタイル（009と共通の不動産向けトークン）

```

## カスタマイズポイント

### 条件の追加・変更
`index.html` 内の `.sf-quick__btn`（エリア・価格帯・間取り）や `.sf-tag`（こだわり条件）を追加・編集してください。`data-factor` は該当件数のデモ計算用の係数（0〜1、小さいほど絞り込みが強い想定）です。実データに接続する場合、この係数ロジックはAPIへの検索リクエスト＆件数取得に置き換えてください。

### 基準件数
`script.js` の `BASE_COUNT`（現在128）が、条件なしの場合の掲載件数です。
