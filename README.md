# ClaudeParts.

地方の中小企業向けWebパーツを、Claudeと一緒に作るチャレンジ。

建設業・医院・学校・不動産など、新潟を中心とした地域の業種に寄り添った
UIコンポーネントを週2〜3本のペースで作って公開していきます。

## 使用技術

- Phase 1：HTML / CSS / JavaScript
- Phase 2：Next.js + Vercel
- Phase 3：microCMS / Supabase（ヘッドレス構成）
