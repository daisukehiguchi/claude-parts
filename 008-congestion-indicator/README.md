# ClaudeParts. 008 / 混雑状況インジケーター

医院・クリニック向けの「混雑状況インジケーター」パーツです。待合室の混雑状況を「空いてます／やや混雑／混雑中」の3段階で表示します。

デザインは005〜007番と同じ医院向けトークン（清潔感のあるティール×白ベース、Noto Sans JP）を継承しています。

## 特徴

- **3段階ステータス表示**：空いてます（ティール）／やや混雑（アンバー）／混雑中（コーラル）で色分け
- **混雑レベルゲージ**：3セグメントのバーでレベルを視覚化
- **待ち人数・待ち時間の表示**：ステータスに連動して数値が切り替わる
- **最終更新表示**：状態が切り替わるたびに更新
- **デモ用切り替えボタン**：カード下部のボタンで3状態をその場で確認可能（実運用では受付システム等のAPIから取得した値で `applyStatus()` を呼び出す形に置き換えてください）

## ファイル構成

```
008-congestion-indicator/
├── index.html   … マークアップ
├── style.css    … スタイル（005〜007と共通の医院向けトークン）

└── README.md
```

## カスタマイズポイント

### 実データへの接続
`script.js` の `statusData` オブジェクトが3状態のラベル・待ち人数・待ち時間の定義です。実運用時は、受付システムやレジ待ち管理システムから取得した人数をもとに、しきい値判定で `applyStatus('empty' | 'busy' | 'full')` を呼び出す形に置き換えてください。デモ用のボタン（`.ic-demo`）はそのまま残しても、削除して自動更新のみにしても構いません。

### しきい値・文言
待ち人数の目安（2人／6人／12人など）や待ち時間、案内文はすべて `statusData` 内の値なので、クリニックの実情に合わせて調整してください。

### 色・フォント
005〜007番と同じ `:root` トークンを使用しています。ステータスごとの配色は `style.css` 内の `.ic-card.is-busy` / `.ic-card.is-full` セレクタで定義しています。

```css
--bg: #eef4f3;
--surface: #ffffff;
--primary: #1f7a6c;
--primary-soft: #e1f3ef;
--amber: #c98a3d;
--amber-soft: #faf0e0;
--coral: #c4665c;
--coral-soft: #f6e8e6;
--ink: #1f2a28;
--muted: #6d7d7a;
--line: #e2eae8;
```
