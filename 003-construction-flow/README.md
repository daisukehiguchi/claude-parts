# ClaudeParts. 003 - 施工の流れ・ステップ表示

地方の中小企業（今回は建設業）向けWeb UIパーツを、Claudeと協働制作するチャレンジ「ClaudeParts.」の第3弾です。

「お問い合わせ」から「アフターフォロー」までの施工フローを、現場の工程表・測量野帳をモチーフにしたステップタイムラインとして表示します。

## デモ

`index.html` をブラウザで開くだけで確認できます。

## コンセプト

- **測量杭ノード**：各ステップ番号を、現場の測量杭に見立てた円形マーカーで表示
- **寸法線レール**：左側の破線は設計図の寸法線をイメージ。スクロールに合わせて青焼き色（blueprint）のラインが伸び、施工の進捗を可視化
- **検査済みハンコ**：各ステップがビューポートに入ると、001（施工実績カード）と同じ意匠の赤い判子（受付・調査・設計・契約・着工・検査・完了・継続）が回転しながら押される
- **最終ステップ（完成・お引き渡し）**は杭ノードが赤に切り替わり、節目として強調

数字のステップ表示は「実際に順序が意味を持つ工程（工程表）」である今回の内容にのみ採用しています。

## 使用しているデザイントークン

ClaudeParts.共通のトークンセットを使用しています（`style.css` の `:root` で定義）。

| 変数 | 用途 | カラー |
|---|---|---|
| `--paper` | 背景（クラフト紙） | `#f1ead9` |
| `--blueprint` | アクセント（青焼き） | `#1c3a5c` |
| `--stamp-red` | ハンコ・強調 | `#b13a2b` |
| `--mustard` | ラベル・タグ | `#d3a545` |
| `--steel` | 補助テキスト・線 | `#6d7377` |

フォントは見出しに Shippori Mincho、本文に Zen Kaku Gothic New を使用（Google Fonts）。

## 挙動・技術メモ

- `IntersectionObserver` でステップの表示を検知し、`.in-view` クラス付与でフェードイン＋ハンコ演出
- 進捗レール（`.rail-fill`）の高さは、最後に表示されたステップの位置から動的に計算
- `prefers-reduced-motion` に対応し、モーション低減設定時はアニメーションを無効化
- 外部ライブラリなし（Vanilla HTML/CSS/JS）
- 520px以下のモバイル幅でノード・スタンプのサイズを縮小、ヘッダーを縦積みに変更

## カスタマイズ

`index.html` 内の各 `.step` ブロックを編集することで、業種・工程数に応じて増減できます。

- ステップ数を変える場合、`.step` を複製・削除するだけでOK（JS側の対応不要）
- ハンコの文言は `.stamp svg text` の中身を変更
- 最終ステップの強調スタイルを外したい場合は `class="step final"` の `final` を削除

## 業種展開の目安

学校・医院・不動産・通信など、他業種の「問い合わせ→提案→契約→実施→フォロー」系フローにも流用可能です（ラベルと本文を差し替えるだけ）。
